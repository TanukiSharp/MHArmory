Text-based weapon set bonus selection Some weapons count as a piece to an armor set bonus. In the armor set bonus selection windows you can select this set bonus. It is possible to use the search bar to quickly toggle the armor set bonus selection.
When text is typed in the search bar, only matching results remain. For example if attack is typed, all armor set bonuses containing attack in their name, their description or the description of the abilities it grants, will remain in view.
